# Мобильная оптимизация системы

## Версия: 3.1.0

## Выполненные оптимизации

### 1. Адаптивный Sidebar
- **Мобильные устройства (< 1024px):**
  - Sidebar скрыт по умолчанию
  - Открывается по кнопке гамбургер в header
  - Показывается как overlay с затемнением фона
  - Автоматически закрывается при клике на пункт меню или вне sidebar

- **Планшеты и десктоп (≥ 1024px):**
  - Sidebar всегда видим
  - Можно сворачивать/разворачивать кнопкой
  - Не перекрывает контент

### 2. Адаптивные Breakpoints
Используются стандартные Tailwind breakpoints:
- `sm:` - 640px и выше (планшеты в портретной ориентации)
- `md:` - 768px и выше (планшеты)
- `lg:` - 1024px и выше (десктоп)

### 3. Оптимизация компонентов

#### Dashboard
- Карточки статистики: `grid-cols-1 sm:grid-cols-2 lg:grid-cols-4`
- Графики: адаптивная высота `h-64 sm:h-80`
- Отступы: `p-4 sm:p-6`, `gap-4 sm:gap-6`

#### TechSpecs
- Заголовки: `text-xl sm:text-2xl lg:text-3xl`
- Отступы: `space-y-4 sm:space-y-6 lg:space-y-8`
- Код блоки: `text-xs sm:text-sm` с горизонтальным скроллом

#### MobileApp
- QR код: адаптивный размер `max-w-full w-full sm:w-auto`
- Сетка: `grid-cols-1 lg:grid-cols-2`
- Текст: `text-sm sm:text-base`

#### EquipmentManagement
- Форма: `grid-cols-1 sm:grid-cols-2`
- Кнопки: адаптивная ширина `w-full sm:w-auto`
- Карточки: `grid-cols-1 sm:grid-cols-2 lg:grid-cols-3`

### 4. Viewport и мета-теги
```html
<meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=5, user-scalable=yes" />
```

### 5. CSS оптимизации
- Адаптивные размеры шрифтов для мобильных
- Отключение tap highlight для лучшего UX
- Оптимизация скроллбаров

### 6. Исправления наложения
- Добавлен `overflow-hidden` для контейнеров
- Использован `flex-wrap` где необходимо
- Правильные `z-index` для overlay
- `whitespace-nowrap` для предотвращения переноса текста в sidebar
