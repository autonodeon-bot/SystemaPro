# Инструкция по установке APK

## Проблема: "Пакет поврежден"

Если при установке APK появляется ошибка "не установлен, т.к. его пакет поврежден", выполните следующие шаги:

### 1. Удалите старую версию приложения
- Перейдите в **Настройки** → **Приложения**
- Найдите приложение **"ЕС ТД НГО"**
- Нажмите **Удалить**
- Подтвердите удаление

### 2. Разрешите установку из неизвестных источников
- Перейдите в **Настройки** → **Безопасность** (или **Безопасность и конфиденциальность**)
- Включите **"Установка из неизвестных источников"** или **"Установка приложений"**
- Для Android 8.0+: разрешите установку для конкретного приложения (файлового менеджера или браузера)

### 3. Проверьте APK файл
- Убедитесь, что файл полностью загружен (размер: ~52.8 MB)
- Попробуйте скачать файл заново
- Проверьте, что на устройстве достаточно свободного места (минимум 100 MB)

### 4. Установите APK
- Откройте файл `app-release.apk` через файловый менеджер
- Нажмите **"Установить"**
- Дождитесь завершения установки

### 5. Альтернативный способ установки через ADB
Если установка через файловый менеджер не работает, используйте ADB:

```bash
adb install -r app-release.apk
```

### 6. Выбор правильного APK для вашего устройства

**Важно:** Используйте APK, соответствующий архитектуре вашего устройства:

- **`app-arm64-v8a-release.apk` (21.0 MB)** - для большинства современных Android устройств (рекомендуется)
- **`app-armeabi-v7a-release.apk` (18.7 MB)** - для старых устройств (32-bit ARM)
- **`app-x86_64-release.apk` (22.3 MB)** - для эмуляторов и некоторых планшетов (64-bit x86)

**Как определить архитектуру устройства:**
1. Установите приложение "CPU-Z" или "Device Info HW"
2. Или используйте ADB команду: `adb shell getprop ro.product.cpu.abi`

**Рекомендация:** Начните с `app-arm64-v8a-release.apk` - он подходит для 95% современных устройств.

## Текущая версия
- **Версия:** 1.0.1+2
- **Размер APK (универсальный):** ~52.8 MB
- **Размер APK (по архитектурам):**
  - arm64-v8a: 21.0 MB (рекомендуется)
  - armeabi-v7a: 18.7 MB
  - x86_64: 22.3 MB
- **Минимальная версия Android:** Зависит от minSdkVersion в build.gradle

## Проверка целостности APK
Для проверки целостности APK можно использовать:

```bash
# Проверка подписи
jarsigner -verify -verbose -certs app-release.apk

# Или через aapt (Android Asset Packaging Tool)
aapt dump badging app-release.apk
```
